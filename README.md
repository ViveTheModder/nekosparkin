# nekosparkin
An Ultimate Battle Editor that supports DBZ Budokai Tenkaichi 2 & 3, both PS2 and Wii versions.

# Walkthrough
As a prerequisite, this tool requires the **parameter containers (in the form of DAT files) to be extracted** from other programs that can **unpack and decompress** the contents of any (Ultimate Battle menu) **PAK file**, such as [HiroTex's SpikeSoft](https://github.com/HiroTex/SpikeSoft/) and [jagger1407's Pakman](https://github.com/jagger1407/Pakman).

Please refer again to the tools listed above, after **applying changes to the parameter containers**, so that the changes can be **carried over to the PAK files** (via **repacking**).

## Select Mode
Regardless of the tool used, Nekosparkin' will always detect 2 DAT files **purely based on their file sizes**, which is **determined by the selected game mode and game version**.
